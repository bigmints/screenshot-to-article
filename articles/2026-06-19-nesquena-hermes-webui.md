## 💼 LinkedIn Post

**Hermes WebUI: 14.7K Stars and the Best Way to Run Hermes From a Browser**

Shubham Saboo shared it on LinkedIn — "Someone just open-sourced a full web UI for Hermes Agent." And it turns out, that someone was **nesquena**, and the project already has **14,709 GitHub stars**.

**What makes it special:**

**1. Truly lightweight — no build step**
Most web UIs for AI agents ship with elaborate frontend frameworks, build pipelines, and npm dependencies. This one is Python + vanilla JS. No bundler, no build step, no framework. Clone and run.

**2. Full CLI parity**
Everything you can do from a terminal — sessions, models, profiles, approvals, workspace browsing, multi-agent orchestration — you can do from the browser. It's not a stripped-down mobile view; it's a first-class interface.

**3. Three-panel power**
Left sidebar for sessions and navigation, center for chat, right for workspace file browsing. Model controls live in a composer footer that's always visible. A circular context ring shows token usage at a glance.

**4. Built for remote access**
Single command to start. Single SSH tunnel command for access from your phone or laptop. Works with Tailscale. Password-protected. Docker-ready.

The agent community has been asking for this since day one. The CLI is powerful, but sometimes you want a browser — especially on mobile, or when sharing a workspace with teammates.

This project has hit a nerve. 14.7K stars in ~3 months says everything.

👉 github.com/nesquena/hermes-webui
